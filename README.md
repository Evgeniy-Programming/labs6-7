# Файловый трансфер TCP/UDP
Проект представляет собой полнофункциональное приложение для передачи файлов по сетям TCP и UDP с графическим интерфейсом. Приложение поддерживает как надежную передачу через TCP, так и быструю через UDP с возможностью выбора протокола в зависимости от требований.
# Требования
-Python 3.7+
-PyQt5
-Стандартные библиотеки Python
# Архитектура
Графическое приложение (visual.py)
TransferApp - Главное окно приложения
TCPServerThread - Поток для TCP сервера
UDPWorker - Поток для UDP сервера
FileTransferThread - Поток для отправки файлов
ReceivedFilesModel - Модель данных для таблицы файлов

# Серверы

TCP - Использует потоковые сокеты, гарантированная доставка
UDP - Использует датаграммные сокеты, быстрая передача

# Клиенты

Асинхронная отправка с индикацией прогресса
Автоматические повторные попытки при ошибках
Подтверждение получения данных

# Протоколы передачи

TCP Протокол

Установка соединения
Отправка заголовка (размер + имя файла)
Потоковая передача данных
Подтверждение получения
UDP Протокол

Отправка метаданных (пакет типа 1)
Подтверждение метаданных
Отправка чанков данных (пакет типа 2)
Подтверждение каждого чанка
Сигнал завершения (пакет типа 3)

# Управление файлами
Автоматические папки

server_downloads/ - для TCP файлов
received_files/ - для UDP файлов

Функции управления
Обновление списка каждые 3 секунды
Открытие папок в системе
Удаление выбранных файлов
Информация о размере и дате

# Запуск

python visual.py
